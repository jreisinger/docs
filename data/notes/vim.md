# Vim

Useful configuration options:

```
:set tabstop=4 shiftwidth=4 expandtab
:set nofoldenable
:set textwidth=0
:syntax off
```

How to search:

```
:grep -iR what .
:cw
```

```
:vimgrep /PATTERN/ FILE(s)
:cnext
:cprex
:cfnext
:cfprev
```

Know your alphabet:

```
:help normal-index
:help insert-index
```

Scroll inside `:term`: `Ctrl-w N`
